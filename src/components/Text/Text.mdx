# Text

The Text component renders a container around text.

## Default Text

The Text component renders a `p` tag by default.

<Canvas of={Stories.Default} />

## Styled Text

The Text component accepts an optional `sx` prop for styling.

<Canvas of={Stories.StyledText} />

## HTML Tags

The Text component accepts an `as` prop to declare the HTML tag used. In the below example Text is assigned `as="a"` along with `href` and `target` props to make the text a styled anchor.
