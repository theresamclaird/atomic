# Text

The `Text` component renders a container around text and like the `Box` it can accept both `as` and `sx` props. The `as` prop defaults to the `p` tag if not provided.

## Import

```jsx
import { Text } from '@theresamclaird/atomic';
```

## Simple Text

```jsx
<Text>This is a themed p element.</Text>
```

## Styling

```jsx
<Text sx={{ padding: '1rem', backgroundColor: '#bdb5d5' }}><Text>This text has padding and background color.</Text></Text>
```

## HTML Tags

```jsx
<Box as="ul">
  <Text as="li">First Bullet</Text>
  <Text as="li">Second Bullet</Text>
  <Text as="li">Third Bullet</Text>
  <Text as="li">
        <Text as="a" href="https://www.mclairdfamily.com/">This is a link.</Text>
        Note the <Text as="span" sx={{ fontWeight: '900' }}>as</Text> and <Text as="span" sx={{ fontWeight: '900' }}>href</Text> props on the anchor.
  </Text>
</Box>
```
