# Box

The `Box` is the most foundational component in the library, and is used to render themeable HTML tags using the optional `as` prop, which defaults to a `div` if not provided.

## Import

```jsx
import { Box } from '@theresamclaird/atomic';
```

```jsx
<Box>This is a div element.</Box>
```

## Styling

```jsx
<Box sx={{ padding: '1rem', backgroundColor: '#bdb5d5' }}>This text is inside a box with some basic styling.</Box>
```

## HTML Tags

```jsx
<Box as="ul">
  <Text as="li">First Bullet</Text>
  <Text as="li">Second Bullet</Text>
  <Text as="li">Third Bullet</Text>
</Box>
```
