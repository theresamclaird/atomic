# Box

The Box is the most foundational component in the library, and is used to render themeable HTML tags.

## Default Box

It renders a `div` tag by default.

<Canvas of={Stories.Default} />

## Styling

Provide an `sx` prop for styling.

<Canvas of={Stories.StyledBox} />

## HTML Tag Declaration

Provide an `as` prop to declare the HTML tag used to render the Box.

<Canvas of={Stories.UnorderedList} />
