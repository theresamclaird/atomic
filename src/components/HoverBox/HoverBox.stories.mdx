# HoverBox

Simple abstraction to render a `Box` according to a hover state passed in a render prop (children).

## Import

```jsx
import { HoverBox } from '@theresamclaird/atomic';
```

## Usage

```jsx
<HoverBox>
  {(mouseEntered) => (
    <Box sx={{ p: 'small', bg: mouseEntered ? '#f00' : 'unset' }}>
      <Text>This text will change the background color to red on hover.</Text>
    </Box>
  )}
</HoverBox>
```
