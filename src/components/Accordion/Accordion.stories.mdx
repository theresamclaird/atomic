# Accordion

## Import

```jsx
import { Accordion } from '@theresamclaird/atomic';
```

## Usage

### Required Props

<Box as="ul">
  <Text as="li">label: Content that is displayed in both the open and closed states.</Text>
  <Text as="li">children: The content rendered in the open state.</Text>
</Box>

```jsx
<Accordion label={<Text as="h3">Lorem Ipsum</Text>}>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Maecenas finibus magna urna, sit
    amet dapibus nisi vehicula quis. Nullam non velit at purus ornare viverra at ut nisl. Donec
    quis dapibus massa, non venenatis ipsum.
  </Text>
</Accordion>
```
