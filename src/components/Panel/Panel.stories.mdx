# Panel

The `Panel` component renders vertical boxes with consistent gutters.

## Import

```jsx
import { Panel } from '@theresamclaird/atomic';
```

```jsx
<Flex
  sx={{
    p: 0,
    m: 0,
    bg: 'background',
    flexDirection: 'column',
    justifyContent: 'flex-start',
    alignItems: 'center',
  }}
>
  <Panel>
    <Text sx={{ p: 'small' }}>This is a panel.</Text>
  </Panel>
  <Panel>
    <Text sx={{ p: 'small' }}>This is another panel.</Text>
  </Panel>
  <Panel>
    <Text sx={{ p: 'small' }}>A Panel contains content and provides standardized widths and gutters.</Text>
  </Panel>
</Flex>
```
