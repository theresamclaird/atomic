# ThemeProvider

## Usage

```jsx
<ThemeProvider>
      <Box sx={{ padding: 1, backgroundColor: 'background' }}>
        <Text>The padding and background color of this Box will reflect the default theme.</Text>
      </Box>
</ThemeProvider>
```

## Overriding the default Theme

```jsx
<ThemeProvider theme={{
      space: [0, '5rem'],
      colors: { background: '#562cac', white: '#fff' },
      text: { p: { fontFamily: 'cursive', color: '#fff' } },
}}>
      <Box sx={{ padding: 1, backgroundColor: 'background' }}>
        <Text>The padding and background color of this Box will reflect the override theme.</Text>
      </Box>
</ThemeProvider>
```
