# useAsync

The useAsync hook provides loading and error indicators for async requests.

## Basic Usage

```jsx
const MyComponent = () => {
  const { loading, value } = useAsync(() => {
    return new Promise((resolve) => {
      setTimeout(() => {
        resolve('Resolved after 5 seconds.');
      }, 5000);
    });
  });

  if (loading) {
    return <p>Loading...</p>;
  }

  return <p>{value}</p>;
};
```
