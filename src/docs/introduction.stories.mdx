# Atomic Component Library

Atomic is a React component library consisting of

- Atoms: Simple React components
- Molecules: More complex components built from atoms
- Modules: JavaScript modules
- Themes: All components within Atomic are themed using Styled System
- Hooks: Simple React hooks

I built these components for fun and to use on my own projects. They're released as open source under the MIT license — free for anyone to use, modify, and distribute. As with any MIT-licensed software, they're provided as-is, without warranty.

## Repository

<a href="https://github.com/theresamclaird/atomic">Code is on github.</a>

## Installation

Published on the public npm registry under the MIT license — no access token required.

```bash
npm install @theresamclaird/atomic
```

`react`, `react-dom`, and `prop-types` are peer dependencies, so install them alongside it if your project doesn't already have them.

## Repository Hygiene

Coverage thresholds are set to 80%, and both testing and linting are run in a husky pre-commit hook.

## Publishing

Releases publish automatically to the public npm registry via GitHub Actions
(trusted publishing / OIDC — no tokens stored) whenever a GitHub Release is
created:

```bash
npm version patch
git push && git push --tags
```

Then draft a GitHub Release from the new tag, and the workflow builds, tests,
and publishes the package.

### Adding New Components

Use plop to add and remove components:

```bash
npm run plop
```

Follow the (intuitive) prompts to add and delete components:

```bash
? [PLOP] Please choose a generator. (Use arrow keys)
❯ addComponent - New Component
  removeComponent - remove component
```
