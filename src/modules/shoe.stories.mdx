# getShoe()

The `getShoe()` function returns a shoe consisting of 1 or more decks of standard playing cards, as an array of card objects. It accepts the number of decks as an argument.

## Use

```jsx
import { getShoe } from '@theresamclaird/atomic';

const shoe = getShoe(6);

```
