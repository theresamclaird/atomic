# shuffle()

The `shuffle()` function shuffles the order of cards in an array using the <a href="https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle" target="_blank">Fisher Yates</a> algorithm.

## Use

```jsx
import { getDeck, shuffle } from '@theresamclaird/atomic';

const deck = getDeck();
const shuffledCards = shuffle(deck);
```
